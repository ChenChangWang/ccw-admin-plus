# ccw-admin-plus

#### Description
Ccw Admin Plus开箱即用的中后台框架/设计解决方案，基于Vue3、Vite、Pinia、Sass等最新技术栈开发，针对不同屏幕大小设计，可配置主题及暗黑主题适配，在线切换，内置实用权限管理设计方案，内置业内通用国际化解决方案，简单配置实现多国语言切换，等等
